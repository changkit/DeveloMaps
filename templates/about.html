About

DeveloMaps is a free collaboration tool for service-oriented organizations. The beta version of the application will provide a slick interface to a rich database of projects. Individuals (e.g. students) can use the site to search for and contact projects of interest. Organizations can use Develomaps to put themselves on the map, so to speak, so that students, donors, and other organizations can reach out to them.

The main goals of DeveloMaps are to:

  • Foster collaboration between service-oriented organizations
  • Provide an accessible, rich, and updated database of service project information
  • Provide a replacement for antiquated project search tools (e.g. Maize Pages)
